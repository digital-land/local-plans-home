Guidance

For planning teams: Access automated email responses for local plan consultations

Learn how to access and update a database generated by the Microsoft Outlook automation.

From:
Department for Levelling Up, Housing and Communities
Published
23 August 2013

Contents

  1. Access and use the database
  2. Export your database
  3. Updating your exported database

Working with your IT department, you should have been able to install the email automation.

Now you can find out how to access the email response database, export the data, and update it if you need to.

Learn more about the automation, including what it does and who can use it.

Access and use the database

If you’ve successfully tested the automation and everything is working properly, you should now be able to use the database.

To get started:

  • go to your Sharepoint URL
  • select Settings – the gear icon – and then select Site contents
  • there will be a folder called ‘Planning Consultation Emails’ –  select this folder to open it
  • there will be a sub-folder with the name of the list you created during the set-up process. All emails are stored in this folder
  • update the status of any downloaded email by selecting the individual row the email is on
  • a new panel will appear with all the details of that specific email, and you can edit the status from this panel

Export your database

Once you've accessed your spreadsheet, you need to export it before you can use it.

To do this:

  • go to your Sharepoint site and select Export at the top of the page
  • choose Export to Excel and download the file
  • open your file in Excel

You’ll be able to see most of the email data, including the location in Sharepoint where the email is saved, but you will not be able to see any attachment data except the number of files that were attached.

Updating your exported database

Even after you've exported the database, you can still pull new emails through to it by following these steps:

  • open your downloaded Excel spreadsheet
  • select the Data tab at the top of the Excel spreadsheet
  • select Refresh all. This will bring any new emails that are in the database through to your Excel spreadsheet
  • select Save once you’ve done this so you do not lose any data
